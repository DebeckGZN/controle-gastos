Configure seu Client ID

Para sincronizar com o Google Drive, abra o arquivo app.js e substitua a linha do CLIENT_ID com o seu.

1
Acesse console.cloud.google.com e crie um projeto
2
Vá em APIs → Credenciais → Criar ID do cliente OAuth 2.0
3
Tipo: Aplicativo da Web. Adicione a URL do seu app em "Origens autorizadas"
4
Ative a Google Drive API na biblioteca de APIs
5
Cole o Client ID no topo do app.js
